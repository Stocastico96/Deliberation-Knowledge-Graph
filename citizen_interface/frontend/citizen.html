How this interface works
Semantic Search

Finds deliberative processes by meaning, not just keywords, using embedding similarity. Filter by platform, contributions, and sort order.

Topics Explorer

Auto-generated topic clusters from the full dataset, with key terms, contribution counts, cross-platform breakdown, and interactive charts.

Arguments Analysis coming soon

Will extract pro/con structures, map reasoning networks and detect logical fallacies via NLP.

Knowledge Graph

Unified RDF dataset: Decide Madrid, Decidim Barcelona, EU Have Your Say, European Parliament debates, aligned to the Deliberation Ontology.
